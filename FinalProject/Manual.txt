실행 방법

(ㄱ)	 GitHub내의 폴더를 다운 받은 후 솔루션 파일(sln 파일)을 실행시켜 솔루션 탐색기 내의 소스파일에서 실행하고자 하는 cpp파일 만 빌드에서 제외시키지 않고 나머지 파일 제외시켜 실행하도록 한다.
(ㄴ)	 EXE파일을 실행하는 방법으로 폴더내의 솔루션 파일이 있는 곳의 Release폴더 안의 exe파일을 실행 시킨다. ‘Final1.exe’, ‘Final2.exe’, ‘Final3.exe’ 파일은 각각 “소스.cpp”, “소스헤더줄임.cpp”, “최소시간.cpp”의 실행 파일을 의미한다.
(ㄷ)	명령 프롬프트를 사용하는 방법으로 명령어로 디렉토리 지정하는 cd 이후에 다운 받은 폴더의 Release 폴더 위치를 입력한다. 
예를 들어cd C:\ Users\ tmddn\ Documents\ Visual Studio 2017\ Projects\ FinalProject\ Release 으로 표현 된다.


두 가지 방법 모두 검색하고자하는 텍스트 파일들을 C://dataset/ 폴더 안에 위치시킨 것을 기준으로 했다.
